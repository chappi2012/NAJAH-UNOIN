# 🚀 NAJAH Union — Team Platform

## Deploy to Netlify in 3 Steps

### **Step 1: Create a GitHub Repository**

1. Go to [github.com/new](https://github.com/new)
2. **Repository name:** `najah-union` (or anything you want)
3. **Description:** "NAJAH Union — Class Team Platform"
4. **Public** (so Netlify can see it)
5. Click **"Create repository"**

---

### **Step 2: Upload Your Files to GitHub**

Inside your new repo, click **"Add file" → "Upload files"**

Upload these files:
- ✅ `najah-dark-colorful.html` (rename to `index.html`)
- ✅ `README.md` (this file)

Or use Git command line:
```bash
git clone https://github.com/YOUR-USERNAME/najah-union.git
cd najah-union
# Copy index.html (the dark colorful version) into this folder
git add .
git commit -m "Initial commit: NAJAH Union dark mode"
git push origin main
```

---

### **Step 3: Deploy to Netlify**

1. Go to [netlify.com](https://netlify.com)
2. Click **"Sign up"** (use your GitHub account)
3. Click **"New site from Git"**
4. Click **"GitHub"** (authorize Netlify to access your repos)
5. Select your **`najah-union`** repository
6. Click **"Deploy site"** (leave settings as default)

**That's it!** 🎉

In 30 seconds, you'll get a URL like:
```
https://jolly-dragon-abc123.netlify.app
```

## 🎯 After Deployment

### **Set a Custom Domain (Optional)**

1. Go to your Netlify site settings
2. Click **"Domain settings"**
3. Click **"Add custom domain"**
4. Enter your domain (e.g., `najah-union.com`)
5. Follow DNS setup instructions
